Sobre este repositório
Este repositório foi criado como parte do desafio Entrega Intermediária do curso de
Engenharia de Software. O objetivo é organizar meus projetos acadêmicos e pessoais com
boas práticas de versionamento usando Git e GitHub, além de construir um perfil
profissional sólido.
